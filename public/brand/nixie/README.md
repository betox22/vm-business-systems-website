# Nixie Layered Avatar

Animation-ready transparent SVG layers based on the current Nixie concept.

All layers use the same `512x512` canvas and can be stacked directly.

Layers:
- `glow.svg`
- `body.svg`
- `head.svg`
- `left-eye.svg`
- `right-eye.svg`
- `mouth-closed.svg`
- `mouth-open.svg`
- `mouth-smile.svg`
- `left-arm.svg`
- `right-arm.svg`
- `chest-symbol.svg`
- `particles.svg`
- `tail-fluid-extensions.svg`

Preview:
- Open `/public/brand/nixie/preview.html`

Animation states covered:
- idle floating
- blinking
- speaking mouth movement
- listening glow
- thinking pulse
- building particles
- success bounce
- alert shake

The current files are SVG layers. PNG export can be generated later from these aligned SVGs with ImageMagick, Inkscape, Figma, or a build pipeline.
